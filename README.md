# Zero-Shot Text Classification using Transformers

## Overview

This project implements zero-shot text classification using transformer-based pre-trained models. The model can classify text into predefined categories without requiring task-specific training data.

## Features

* Zero-shot text classification
* Multi-label classification
* Flexible category prediction
* Transformer-based NLP
* Pre-trained language models

## Technologies Used

* Python
* Hugging Face Transformers
* Jupyter Notebook
* VS Code

## Project Files

* `app.py` – Main application code
* `requirements.txt` – Project dependencies
* `sample_input.txt` – Example input texts
* `sample_output.txt` – Example predictions
